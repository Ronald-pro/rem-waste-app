
# A revamp design for Skip selection


## Installation
git clone https://github.com/Ronald-pro/rem-waste-app.git  - in your local machine
cd rem-waste-app

#for dependancies
npm install

#running the application
npm run dev

#access the application through:
http://localhost:5173/
